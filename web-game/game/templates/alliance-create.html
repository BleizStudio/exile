[[create]]
Nouvelle alliance
Formez une alliance, dirigez la et imposez-vous dans la galaxie !
Choisissez un nom et un tag d'alliance. Le tag sera affiché à coté des noms des membres de l'alliance, et est en général un diminutif ou l'acronyme du nom de l'alliance.
Par exemple : La Grande Alliance / LGA.

Les frais de création d'alliance s'élèvent à 10 000 crédits.
Nom de l'alliance : [[name_already_used]]
Ce nom est déjà utilisé, veuillez en choisir un autre.[/name_already_used]] [[invalid_name]]
Entrez un nom de 4 à 32 lettres (A-Z, 0-9, -, .).[/invalid_name]]
Tag de l'alliance : [[tag_already_used]]
Ce tag est déjà utilisé, veuillez en choisir un autre.[/tag_already_used]] [[invalid_tag]]
Le tag doit être composé de 2 à 4 lettres.[/invalid_tag]]
Entrez une description qui présente votre alliance, ses objectifs, ses alliés, etc. Cette description peut être modifiée par la suite dans les options de l'alliance.
[/create]] [[cant_create]]
Vous avez récemment quitté une alliance, il vous est actuellement impossible de créer une alliance.
[/cant_create]]
